
            The Javamm Virtual Machine, CMD wrapper and Simple IDE
                            =================

    What is it?
    -----------
    Java-- is the simplified version of Java programming language.
    Please use CMD wrapper (javamm.script) or Simple IDE (ide.script)
    to launch the Javamm Virtual Machine
